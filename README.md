# StyleFusion — Deployment Guide

## 3-Page Structure
- `index.html` — Home (with merged App Screen section)
- `about.html` — About / Brand Story / Values
- `contact.html` — Contact / Demo Request / Inquiry

## Deploy to Cloudflare Pages (Free)
1. Push this folder to a GitHub repository
2. Go to https://pages.cloudflare.com → Create project → Connect Git repo
3. Build settings: **Framework: None** | Build command: *(blank)* | Output directory: `/`
4. Click Save and Deploy — live in ~60 seconds

## Deploy to Netlify (Free)
- Drag and drop this entire folder to https://app.netlify.com/drop

## Deploy to GitHub Pages (Free)
- Settings → Pages → Deploy from branch → main → / (root)

## Before Going Live — Replace These
- `hello@stylefusion.app` → real email in all 3 HTML files + main.js
- `+91 90000 00000` → real phone number
- `assets/logo.png` → your actual logo file
- `assets/stylefusion-app-screen.webp` → real app screenshots
- Testimonial names/quotes → real merchant feedback
- Market statistics → verified data
